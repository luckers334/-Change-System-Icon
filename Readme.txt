Features:

ChangeSystemIcon makes it easy for you to change the computer icon in your Windows 7 "Devices and Printers" page.
ChangeSystemIcon can use information gathered from your systems BIOS or you can manually define the various properties.
A unique hardware ID for your system is automatically calculated by the ChangeSystemIcon program.

The icon to represent your system must meet the following format and size requirements:
	
256x256: 32bit (True-color + Alpha)
  48x48: 32bit (True-color + Alpha)
  32x32: 32bit (True-color + Alpha)
  24x24: 32bit (True-color + Alpha)
  16x16: 32bit (True-color + Alpha)

  48x48: 8bit (256 colors)
  32x32: 8bit (256 colors)
  24x24: 8bit (256 colors)
  16x16: 8bit (256 colors)

  48x48: 4bit (16 colors)
  32x32: 4bit (16 colors)
  24x24: 4bit (16 colors)
  16x16: 4bit (16 colors)


To automatically copy the Device Metadata Package to the Device Metadata Store make sure you check the option.

To manually install the Device Metadata Package to your system's Device Metadata Store Copy the package to the following directory:

%PROGRAMDATA%\Microsoft\Windows\DeviceMetadataStore
